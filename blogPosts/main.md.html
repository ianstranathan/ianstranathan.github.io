**Noodler of Math, Physics and Computer Graphics** Ian Stranathan This site is mostly a personal, organizational resource to act as a linking archive for the various things I become interested in and work on. About ============================================== I think there is a deep difference between knowing something and understanding it. The only way to really understand something is to start from scratch. Unfortunately, anything new is awfully confusing most of the time. Sometimes it helps to write about it and that finds its way here. The landing page is a random set of shaders made for fun & greasing the groove.
