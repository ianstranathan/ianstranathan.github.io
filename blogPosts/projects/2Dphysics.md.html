Physics System =================================================== I made visualization tests or wrote articles for each main component: * Euler Method (Verlet Integration too eventually) for numerical integration to propagate motion * [Seperating Axis Theorem](../../projects/2Dphysics/SATVisualization/index.html) for narrow phase collision detection and contact normal * Impulse Based Collision Response * [Quadtree](../../projects/2Dphysics/quadTreeVisualization/index.html) for broadphase collision detection Resources =================================================== SAT: * SAT tutorial by dyn4j * Tutorial by n++ developer Impulse Based Collision Response: * Ian Millington - Game Physics Engine Development * Randy Gaul - How to Create a Custom 2D Physics Engine: The Basics and Impulse Resolution * Brian Mirtich & John Canny - Impulse-based Simulation of Rigid Bodies Quadtree: * Wikipedia * The Coding Train
